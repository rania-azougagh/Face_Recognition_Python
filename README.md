# Face Recognition System using Python & OpenCV

This project is a real-time face recognition application built with Python and OpenCV. It uses the LBPH (Local Binary Patterns Histograms) algorithm to identify faces.

## Features
* Real-time face detection.
* Data collection script for custom datasets.
* Training model using LBPH.
* Face identification with accuracy percentage.

## Technologies Used
* Python 3
* OpenCV
* NumPy
